## Primeros pasos

- Clonar el proyecto e instalar los paquetes

```sh
git clone https://github.com/mgvictor7/shopping-list-react-firebase.git

cd shopping-list-react-firebase
npm install
```

- Duplicar el fichero /src/config/example.config.js y renombrarlo como config.js

- Añadir en el fichero /src/config/config.js la configuración que se suministrar al crear un proyecto en Firebase [Primeros pasos en Firebase](https://firebase.google.com/docs/web/setup)
